# Projeto 1 - Calculadora

# Calculadora em Shell

Como executar
1. Clonar o repositório
2. Acessar a pasta do projeto:
      cd Projeto 1 - Calculadora
     
4. Dar permissão de execução ao script:
      chmod +x calculadora.sh
      
5. Executar o script:
      ./calculadora.sh
      
6. Alternativa sem alterar permissão:
      bash calculadora.sh

   

# Calculadora em Python

Explicação do código:

  1. Entrada do nome do usuário e exibição de uma mensagem de boas-vindas personalizada.
  2. Loop principal (while True) mantém a calculadora ativa até que o usuário decida sair.
  3. O usuário escolhe o tipo de operação matemática: soma, subtração, multiplicação ou divisão.
  4. O programa solicita dois números, convertidos para float para permitir números decimais.
  5. Execução das operações:
      Soma (1), subtração (2), multiplicação (3) e divisão (4, com verificação para evitar divisão por zero). Qualquer escolha inválida retorna “Operação inválida”.
  6. Repetição: Ao final, o programa pergunta se deseja realizar outra operação. Se a resposta for diferente de "s", o loop termina e a calculadora encerra.


Como usar o programa:
  1. Digite seu nome quando solicitado.
  2. Escolha o tipo de operação digitando o número correspondente (1, 2, 3 ou 4).
  3. Informe os dois números que deseja calcular.
  4. Veja o resultado exibido no terminal.
  5. Decida se quer realizar outra operação (s para sim, n para não).
